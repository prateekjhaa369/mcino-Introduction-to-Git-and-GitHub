# Introduction to Git and GitHub Final Project

## Project Name: Simple Interest Calculator
This project is part of the final assessment for the **Introduction to Git and GitHub** course. It demonstrates core version control skills including branching, merging, and basic shell scripting.

## Description
A simple shell script that calculates the simple interest based on user input for principal, rate of interest, and time period.
